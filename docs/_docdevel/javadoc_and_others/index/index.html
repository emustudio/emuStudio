emuLib javadoc

Java documentation (a.k.a. javadoc) exists for emuLib project, which is a run-time library of emuStudio and provides the API for plug-ins. It is a documentation reference for plug-in developers.

It can be downloaded from this link. Alternatively, open it in the new window: emuLib javadoc online.
